เกี่ยวกับฉัน

นักวิชาการคอมพิวเตอร์ที่มีประสบการณ์ด้านการจัดการข้อมูลและระบบสารสนเทศของหน่วยงาน สามารถเขียน SQL เพื่อวิเคราะห์และจัดทำรายงานตามตัวชี้วัด พร้อมดูแลระบบงานด้านสาธารณสุข มีความเข้าใจในงานราชการและกระบวนการจัดซื้อจัดจ้าง รวมถึงมีทักษะในการประสานงานและสนับสนุนงานไอทีให้ดำเนินได้อย่างมีประสิทธิภาพ

ตำแหน่ง : นักวิชาการคอมพิวเตอร์

  • เขียน SQL จัดทำรายงานตามตัวชี้วัด
  • จัดทำโครงการจัดซื้อจัดจ้างครุภัณฑ์คอมพิวเตอร์และระบบสารสนเทศของหน่วยงาน
  • ดูแลและบริหารจัดการระบบสารสนเทศของกระทรวงสาธารณสุข (MOPH) ได้แก่ MOPH Appointment, หมอพร้อม Station, Telemedicine, GTW Backoffice และ Smart Hospital รวมถึงระบบอื่น ๆ ที่เกี่ยวข้อง
  • IT Support และประสานงานระบบต่าง ๆ

ทักษะ

  • SQL (ดึงข้อมูล วิเคราะห์ และสร้างรายงาน)
  • HOSxP XE4, GTW Backoffice, ระบบสารสนเทศของกระทรวงสาธารณสุข (MOPH)
  • Web Development (HTML, CSS, PHP เบื้องต้น)
  • AI Tools สำหรับงาน IT
  • Cybersecurity เบื้องต้น
  • Network เบื้องต้น
